第二步：提出展示需求
需求：

 1. 一个问候语
 2. 导入一个风格图片
 3. 统计一个信以为真的访问量
 4. 展示你的活跃度
 5. 介绍一下自己
 6. 传递你的博客与公众号
 7. 你的工作平台、技能与工具是什么
 8. 你的擅长语言对比
 9. 你日常关注的大佬
