#  gitbook 插件：github

##  1. github 插件
在右上角添加 github 图标。

这里是提供 github 的图标和URL 配置，URL可以根据自己需求配置。

###  1.1 安装

```bash
npm i gitbook-plugin-github
```
### 1.2 配置

```c
{
    "plugins": [ 
        "github" 
    ],
    "pluginsConfig": {
        "github": {
            "url": "https://github.com/Ghostwritten"
        }
    }
}
```
### 1.3 效果
> [!NOTE|style:flat|lable:Mylable|iconVisibility:hidden]
> 请阅览右上角。
