#  Gitbook 插件：TODO

##  1. todo 代办
插件地址：[https://www.npmjs.com/package/gitbook-plugin-todo](https://www.npmjs.com/package/gitbook-plugin-todo)

>  Warning
> 安装时失败 install plugin "todo" (*) from NPM with version 0.1.3

### 1.1 配置

```bash
{
    "plugins": ["todo"]
}
```
### 1.2 效果
Markdown 语法

```bash
* [ ]  write some articles
* [x]  drink a cup of tea
```

渲染效果

[ ] write some articles
[x] drink a cup of tea
